# Конкурс прогнозов СМИ — 2 апреля 2026

## Описание
Скрипт для генерации прогнозов заголовков российских СМИ на 2 апреля 2026 года с использованием GigaChat API (Сбер).

## Прогнозы
Скрипт генерирует 5 прогнозов:
1. **ТАСС** — День единения народов России и Беларуси
2. **РИА Новости** — Старт призывной кампании
3. **Интерфакс** — Итоги эксперимента с электронными повестками
4. **Вести (Россия 24)** — Паводки в Дагестане и Чечне
5. **Интерфакс** — Параметры ОФЗ
